Об авторе

Доброго времени суток! Меня зовут Дмитрий. Я выпускник Веб-факультета Яндекс.Практикума по профессии Веб-разработчик. Этот сайт - моя дипломная работа.

Владею кроссбраузерной и adaptive/responsive версткой (HTML, CSS) интернет приложений по методологии БЭМ. Использую современные возможности языка JavaScript для программирования интерфейса пользователя и динамики сайта в объектно-ориентированной парадигме. Создаю бэкэнд проекта (Node.js, Express.js, MongoDB, Nginx). Настраиваю сборку с помощью Webpack, диплою сайт на удаленный сервер, свободно общаюсь с командной строкой. Применяю различные инструменты для разработки и отладки ПО (VSCode, Git, Figma, dev tools, js-console, Postman). Следую принципам линтинга кода при помощи ESLint и стилю синтаксиса Airbnb. Имею навыки коммуникаций в среде взаимодействия разработчиков Slack. Четко представляю понятие дедлайна.

Обучение в Яндекс.Практикум позволило мне за 9 месяцев освоить и закрепить на уровне мышления огромный багаж знаний. Это очень важно в реалиях быстроразвивающихся технологий разработки ПО и востребованности в веб-специалистах. При старте в профессии Вы наставляете на верный путь, мотивируете, даете новые возможности развития личности. Спасибо всей команде Яндекс.Практикума за отличный продукт!
